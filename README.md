# PocketAI Pro

**App:** PocketAI Pro
**Version:** 1.0.0
**Built by:** [Tateru Pro](https://github.com/ushanboe/tateruProPlus) (AppForge pipeline — Tateru Pro's predecessor)
**Last updated:** 2026-04-25

Offline AI chat assistant for Android — runs **13 on-device LLMs** with vision and voice, no cloud calls, no telemetry, no subscription. Multi-backend support via `fllama` (llama.cpp bindings) and Gemma 4 LiteRT-LM. Resumable model downloads. **Built end-to-end by the Tateru / AppForge pipeline from a feature brief.**

This repo exists as a public **sample app** demonstrating that real on-device LLM inference apps can be built fully autonomously by the Tateru pipeline — no manual coding.

---

## Why this is a useful sample

PocketAI Pro proves the pipeline can ship apps in the **on-device AI** category — historically one of the harder categories because:

- Native plugin combinations (fllama + LiteRT-LM + voice + vision + storage) often have permission and registration quirks
- Large model files require resumable downloads (not just naive HTTP)
- Inference loops need careful timeout / cancellation handling
- The UI must behave well during multi-second inference responses

All of this was generated by the pipeline. The app is **27 Dart files** of real, runnable code.

---

## What's in the box

- **13 on-device LLMs** preconfigured (mix of GGUF models via fllama + Gemma 4 via LiteRT-LM)
- **Vision support** — multimodal models can read images
- **Voice input/output** — speech-to-text in, TTS out
- **Resumable model downloads** — pause/resume large model files without restart
- **Streaming chat** — token-by-token rendering
- **Conversation history** — persisted locally via sqflite
- **Markdown responses** — full markdown rendering of model output
- **Material 3 UI** — animated, themed, polished

## Tech stack (key packages)

`fllama` (llama.cpp bindings) · `provider` (state) · `sqflite` (persistence) · `flutter_tts` + `speech_to_text` (voice) · `image_picker` (vision input) · `flutter_markdown` (output) · `flutter_animate` + `lottie` (motion)

## Where on-device AI sits in Tateru's reference patterns

PocketAI Pro is one of three sample apps that exercise the on-device-AI category, alongside:

- **Pocketminder** — offline AI assistant (Gemma 4 E2B via LiteRT-LM)
- **Nomadical** — offline survival reference with multi-doc PDF RAG (Gemma 4 E2B + standalone FTS5 + syncfusion PDF text extraction)

The Tateru pipeline includes a tested reference recipe for **Flutter on-device RAG** (`agents/bob-the-builder/references/flutter_on_device_rag.md`) so future apps in this category can reuse the validated stack: Gemma 4 E2B model + LiteRT-LM SDK + standalone FTS5 BM25 search + syncfusion PDF text extraction with pdfx + ML Kit OCR fallback for scanned documents.
